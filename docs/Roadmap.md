# Family OS — Roadmap

---

## 当前阶段：Alpha（本地 / Preview）

目标：验证核心体验，确定信息架构，建立设计系统。

---

## Alpha 版本规划

### ✅ Alpha 0.3 — 基础骨架
- 五页结构搭建（今天 / 资产 / 成长 / 家 / 更多）
- 底部导航
- 静态数据展示
- 温暖极简风格确立

### ✅ Alpha 0.4 — Polish Sprint
- Design Token 统一
- 组件库建立（Card / HeroCard / SectionLabel / AdvisorCard）
- 真实折线图
- 数字动画、页面动画、点击反馈
- AI 顾问自然语言化
- 首页信息精简

### 🔲 Alpha 0.5 — 家 Module + 月度回顾
- [x] `/family` 家 Home Module 重设计：Hero / 家庭成员 / Maruko / 公司 / 车辆 / 证件 / 家庭时间线
- [ ] 月度回顾填写页面
- [ ] 字段：净资产变化 / 大事件 / 收入概况 / 一次性支出 / 下月重点 / 最开心的事
- [ ] AI 月报总结（静态文案版）
- [ ] LocalStorage 持久化（月度数据）

### 🔲 Alpha 0.6 — 数据持久化
- 全局 LocalStorage 状态管理
- 资产数据可编辑
- 目标数据可编辑
- 首页数字来自真实存储数据

### 🔲 Alpha 0.7 — Supabase 接入
- 用户认证（邮箱 / Google）
- 云端同步月度数据
- 多设备访问
