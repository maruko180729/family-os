# Family OS — Roadmap

---

## 当前阶段：Alpha（本地 / Preview）

目标：验证核心体验，确定信息架构，建立设计系统。

---

## Alpha 版本规划

### ✅ Alpha 0.3 — 基础骨架
- 五页结构搭建（今天 / 资产 / 成长 / 家 / 更多）
- 底部导航
- 静态数据展示
- 温暖极简风格确立

### ✅ Alpha 0.4 — Polish Sprint
- Design Token 统一
- 组件库建立（Card / HeroCard / SectionLabel / AdvisorCard）
- 真实折线图
- 数字动画、页面动画、点击反馈
- AI 顾问自然语言化
- 首页信息精简

### ✅ Alpha 0.5 — 家庭经营 + 家庭资产
- Sprint 1：`/management` 月度收支录入（工资/配偶/其它 × 固定/信用卡/其它）
- Sprint 2：`/assets` 月度资产快照（4 分类 × SVG 趋势）
- 导航重构：今天 / 经营 / 资产 / 未来 / 家

### ✅ Alpha 0.6 — 家 Module + 月度回顾
- Sprint 3：`/family` 全新设计（成员 / Maruko / 公司 / 车辆 / 证件 / 时间线）
- Sprint 4：`/review` 月度回顾（净资产变化 / 大事件 / 收入 / 支出 / 下月 / 开心）
- Stabilization Sprint：文档同步、数据层清理、首页真实数据联动
- 首页 AI 顾问读取真实收支 + 资产快照数据

### 🔲 Alpha 0.7 — 数据编辑
- 目标数据可编辑（Goals 写入 LocalStorage）
- `/family` 各卡片支持编辑（成员 / 公司 / 车辆 / 证件）
- 资产快照支持单分类修改

### 🔲 Alpha 0.8 — Supabase 接入
- 用户认证（邮箱 / Google）
- 云端同步月度数据
- 多设备访问
